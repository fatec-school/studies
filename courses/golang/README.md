# Go: Zero to Advanced (Udemy)

This directory contains notes, code examples, and exercises from the Udemy course Go: do Zero ao Avançado, which focuses on mastering the Go programming language from fundamental syntax to professional application development. The course provides a deep dive into Go's unique features, emphasizing performance, simplicity, and modern backend development.

## Course Summary

This course covers Go fundamentals and advanced programming patterns, including:

- Core language syntax, data types, and control structures
- Effective use of Pointers, Structs, and Interfaces for clean code
- Advanced Concurrency model using Goroutines and Channels
- Web development: building high-performance REST APIs and working with HTTP
- Database integration and persistence using SQL and modern drivers
- Automated testing, including unit tests, benchmarks, and mocks
- Practical projects to build CLI tools, microservices, and scalable backend systems
- Understanding Go's ecosystem, package management, and best practices
